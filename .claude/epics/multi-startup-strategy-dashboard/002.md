# T-02: Project switcher sidebar + setActiveProject() completo

## Obiettivo
Aggiungere i tab [PN] [MC] in cima alla sidebar e implementare il cambio di contesto completo.
Dopo questo task si può navigare tra i due progetti: header, sezioni e geo cambiano dinamicamente.

## Lavoro da fare

### 1. HTML — Switcher in cima a nav.sidebar
Aggiungere prima del primo .sl "Navigation":
```html
<div class="proj-switch" id="proj-switch"></div>
```

### 2. CSS — Stili switcher (minimo, riusa variabili esistenti)
```css
.proj-switch{display:flex;gap:4px;padding:8px 8px 4px;margin-bottom:4px}
.ps-tab{flex:1;padding:6px 8px;border-radius:var(--r);font-size:12px;font-weight:600;
  cursor:pointer;border:1.5px solid var(--g200);background:white;color:var(--g600);
  text-align:center;transition:all .15s}
.ps-tab.active{border-color:var(--accent);background:var(--accent-l);color:var(--accent)}
```

### 3. JS — Render switcher + setActiveProject() completo
```js
function renderProjectSwitch() {
  document.getElementById('proj-switch').innerHTML = PROJECTS.map(p =>
    `<div class="ps-tab${p.id===activeProject.id?' active':''}" onclick="setActiveProject('${p.id}')">${p.logo}</div>`
  ).join('');
}

function setActiveProject(id) {
  activeProject = PROJECTS.find(p => p.id === id) || PROJECTS[0];
  // CSS vars
  document.documentElement.style.setProperty('--accent', activeProject.color);
  document.documentElement.style.setProperty('--accent-l', activeProject.colorLight);
  document.documentElement.style.setProperty('--accent-m', activeProject.colorMid);
  // Header
  document.querySelector('.h-logo').textContent = activeProject.logo;
  document.querySelector('.h-title').textContent = 'Strategy Dashboard';
  document.querySelector('.h-sub').textContent = activeProject.name + ' · ' + activeProject.subtitle;
  // Sidebar sections visibility
  document.querySelectorAll('[data-project]').forEach(el => {
    const p = el.dataset.project;
    el.style.display = (p === 'all' || p === activeProject.id) ? '' : 'none';
  });
  // Geo filter rebuild
  renderGeo();
  // Refresh switcher active state
  renderProjectSwitch();
  // Refresh active page content
  refreshActivePage();
  // Save preference
  localStorage.setItem('mp_active_project', id);
}

function renderGeo() {
  // rebuild geo section in sidebar with activeProject.geo
}
```

### 4. HTML — Aggiungere data-project agli elementi sidebar
- Sezioni comuni (overview, strat, funnel): `data-project="all"`
- Sezioni PN only (kw, neg): `data-project="pn"`
- Sezioni MC only (retention): `data-project="mc"`
- Mercati: sostituire il blocco statico con `<div id="geo-sidebar"></div>`

### 5. Init al caricamento
```js
// In init() o dopo doLogin()
const savedProject = localStorage.getItem('mp_active_project') || 'pn';
setActiveProject(savedProject);
renderProjectSwitch();
```
