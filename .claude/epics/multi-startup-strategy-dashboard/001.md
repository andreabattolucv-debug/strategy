# T-01: Config system + CSS --accent refactor

## Obiettivo
Creare la base config-driven dell'app senza impatto visivo su Pink Notebook.
Zero regressioni: dopo questo task la dashboard PN deve sembrare e funzionare identica.

## Lavoro da fare

### 1. Sostituire --pink con --accent
Nelle CSS variables in cima al file, aggiungere:
```css
--accent: #E91E8C;        /* default PN, override a runtime */
--accent-l: #FBEAF0;
--accent-m: #F4C0D1;
```
Fare find/replace di `var(--pink)` → `var(--accent)` e `var(--pink-l)` → `var(--accent-l)` etc.
Mantenere --pink come alias per retrocompatibilità.
